WordPress plugin that provides membership features and controls posts based on the membership level.
== Description ==
This plugin can be used to provide some membership Level feature (basic-level, premium level and gold-level) to you wordpress site.Post will be automatically restricted like basic-level member can use 2 post, premium-level member can use 5 posts,and gold-level member can access all the posts without restriction.

== Installation ==
Upload the plugin files to the /wp-content/plugins/plugin-name directory, or install the plugin through the WordPress plugins screen directly.
Activate the plugin through the \'Plugins\' screen in WordPress.
